# Admin Flow

## Overview
The Admin module manages administrative users in the placement system. Admins have full access to manage system operations.

## Entity Model
```
Admin
├── id (Long) - Primary Key, Auto-generated
├── name (String) - Admin's full name
└── password (String) - Admin's password
```

## API Endpoints

### 1. Get All Admins
**Endpoint:** `GET /api/admin`
```
Request: GET http://localhost:8080/api/admin
Response: 
[
  {
    "id": 1,
    "name": "Rajesh Kumar",
    "password": "admin@123"
  },
  ...
]
```

### 2. Create Admin
**Endpoint:** `POST /api/admin`
```
Request: POST http://localhost:8080/api/admin
Body: {
  "name": "New Admin",
  "password": "admin@password"
}

Response:
{
  "id": 6,
  "name": "New Admin",
  "password": "admin@password"
}
```

### 3. Update Admin
**Endpoint:** `PUT /api/admin`
```
Request: PUT http://localhost:8080/api/admin
Body: {
  "id": 1,
  "name": "Rajesh Kumar Updated",
  "password": "newpass123"
}

Response:
{
  "id": 1,
  "name": "Rajesh Kumar Updated",
  "password": "newpass123"
}
```

### 4. Delete Admin
**Endpoint:** `DELETE /api/admin/{id}`
```
Request: DELETE http://localhost:8080/api/admin/1
Response: (No content)
```

## Data Flow

```
┌─────────────────────────┐
│    Frontend/Client      │
└────────────┬────────────┘
             │
             │ HTTP Request
             ▼
┌─────────────────────────┐
│  AdminController        │ (REST Endpoints)
│  @RestController        │
│  @RequestMapping        │
│  @CrossOrigin(*)        │
└────────────┬────────────┘
             │
             │ Calls Service Methods
             ▼
┌─────────────────────────┐
│  AdminService           │ (Business Logic)
│  (Interface)            │
└────────────┬────────────┘
             │
             │ Implements
             ▼
┌─────────────────────────┐
│  AdminServiceImpl        │ (Implementation)
│  @Service               │
│  @Transactional         │
└────────────┬────────────┘
             │
             │ Uses Repository
             ▼
┌─────────────────────────┐
│  AdminRepository        │ (Data Access)
│  extends CrudRepository │
└────────────┬────────────┘
             │
             │ Database Operations
             ▼
┌─────────────────────────┐
│  PostgreSQL Database    │
│  (admin table)          │
└─────────────────────────┘
```

## Database Table

```sql
CREATE TABLE admin (
    id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
    name VARCHAR(255),
    password VARCHAR(255)
)
```
