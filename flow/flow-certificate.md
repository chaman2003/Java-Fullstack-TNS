# Certificate Flow

## Overview
The Certificate module manages certificates issued to students and colleges in the placement system. Certificates track achievements and qualifications by year and institution.

## Entity Model
```
Certificate
├── id (Long) - Primary Key, Auto-generated
├── year (Integer) - Year of certification (required)
└── college (String) - College/Institution name
```

## API Endpoints

### 1. Get All Certificates
**Endpoint:** `GET /api/certificate`
```
Request: GET http://localhost:8080/api/certificate
Response: 
[
  {
    "id": 1,
    "year": 2024,
    "college": "IIT Delhi"
  },
  ...
]
```

### 2. Create Certificate
**Endpoint:** `POST /api/certificate`
```
Request: POST http://localhost:8080/api/certificate
Body: {
  "year": 2024,
  "college": "IIT Bombay"
}

Response:
{
  "id": 6,
  "year": 2024,
  "college": "IIT Bombay"
}
```

### 3. Update Certificate
**Endpoint:** `PUT /api/certificate`
```
Request: PUT http://localhost:8080/api/certificate
Body: {
  "id": 1,
  "year": 2025,
  "college": "IIT Delhi Updated"
}

Response:
{
  "id": 1,
  "year": 2025,
  "college": "IIT Delhi Updated"
}
```

### 4. Delete Certificate
**Endpoint:** `DELETE /api/certificate/{id}`
```
Request: DELETE http://localhost:8080/api/certificate/1
Response: (No content)
```

## Data Flow

```
┌─────────────────────────┐
│    Frontend/Client      │
└────────────┬────────────┘
             │
             │ HTTP Request
             ▼
┌─────────────────────────┐
│  CertificateController  │ (REST Endpoints)
│  @RestController        │
│  @RequestMapping        │
│  @CrossOrigin(*)        │
└────────────┬────────────┘
             │
             │ Calls Service Methods
             ▼
┌─────────────────────────┐
│  CertificateService     │ (Business Logic)
│  (Interface)            │
└────────────┬────────────┘
             │
             │ Implements
             ▼
┌─────────────────────────┐
│  CertificateServiceImpl  │ (Implementation)
│  @Service               │
└────────────┬────────────┘
             │
             │ Uses Repository
             ▼
┌─────────────────────────┐
│  CertificateRepository  │ (Data Access)
│  extends JpaRepository  │
└────────────┬────────────┘
             │
             │ Database Operations
             ▼
┌─────────────────────────┐
│  PostgreSQL Database    │
│  (certificate table)    │
└─────────────────────────┘
```

## Database Table

```sql
CREATE TABLE certificate (
    id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
    year INTEGER NOT NULL,
    college VARCHAR(255)
)
```
