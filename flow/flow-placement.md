# Placement Flow

## Overview
The Placement module manages company placements and job opportunities available for students. It tracks placement information including company name, required qualifications, and year of placement.

## Entity Model
```
Placement
├── id (Long) - Primary Key, Auto-generated
├── name (String) - Company/Placement name
├── qualification (String) - Required qualification (e.g., B.Tech, M.Tech)
└── year (Integer) - Year of placement (required)
```

## API Endpoints

### 1. Get All Placements
**Endpoint:** `GET /api/placement`
```
Request: GET http://localhost:8080/api/placement
Response: 
[
  {
    "id": 1,
    "name": "Infosys",
    "qualification": "B.Tech",
    "year": 2024
  },
  ...
]
```

### 2. Create Placement
**Endpoint:** `POST /api/placement`
```
Request: POST http://localhost:8080/api/placement
Body: {
  "name": "Microsoft",
  "qualification": "B.Tech/M.Tech",
  "year": 2025
}

Response:
{
  "id": 6,
  "name": "Microsoft",
  "qualification": "B.Tech/M.Tech",
  "year": 2025
}
```

### 3. Update Placement
**Endpoint:** `PUT /api/placement`
```
Request: PUT http://localhost:8080/api/placement
Body: {
  "id": 1,
  "name": "Infosys Limited",
  "qualification": "B.Tech/B.E",
  "year": 2025
}

Response:
{
  "id": 1,
  "name": "Infosys Limited",
  "qualification": "B.Tech/B.E",
  "year": 2025
}
```

### 4. Delete Placement
**Endpoint:** `DELETE /api/placement/{id}`
```
Request: DELETE http://localhost:8080/api/placement/1
Response: (No content)
```

## Data Flow

```
┌─────────────────────────┐
│    Frontend/Client      │

└────────────┬────────────┘
             │
             │ Calls Service Methods
             ▼
┌─────────────────────────┐
│ PlacementService        │ (Business Logic)

└────────────┬────────────┘
             │
             │ Implements

┌─────────────────────────┐

             │

┌─────────────────────────┐

└────────────┬────────────┘

             ▼
```

## Database Table

```sql
CREATE TABLE placement (
    id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
    name VARCHAR(255),
    qualification VARCHAR(255),
    year INTEGER NOT NULL
)
```
