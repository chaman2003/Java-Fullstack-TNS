# Student Flow

## Overview
The Student module manages information about students in the placement system. Students can be created, retrieved, updated, and deleted.

## Entity Model
```
Student
├── sid (Integer) - Primary Key, Auto-generated
├── name (String) - Student's full name
├── collegeName (String) - Name of the college
└── phoneNo (Long) - Student's phone number
```

## API Endpoints

### 1. Get All Students
**Endpoint:** `GET /api/student`
```
Request: GET http://localhost:8080/api/student
Response: 
[
  {
    "sid": 1,
    "name": "Amit Verma",
    "collegeName": "IIT Delhi",
    "phoneNo": 9876543210
  },
  ...
]
```

### 2. Create Student
**Endpoint:** `POST /api/student`
```
Request: POST http://localhost:8080/api/student
Body: {
  "name": "John Doe",
  "collegeName": "IIT Delhi",
  "phoneNo": 9876543210
}

Response:
{
  "sid": 6,
  "name": "John Doe",
  "collegeName": "IIT Delhi",
  "phoneNo": 9876543210
}
```

### 3. Update Student
**Endpoint:** `PUT /api/student`
```
Request: PUT http://localhost:8080/api/student
Body: {
  "sid": 1,
  "name": "Amit Verma Updated",
  "collegeName": "IIT Bombay",
  "phoneNo": 9876543211
}

Response:
{
  "sid": 1,
  "name": "Amit Verma Updated",
  "collegeName": "IIT Bombay",
  "phoneNo": 9876543211
}
```

### 4. Delete Student
**Endpoint:** `DELETE /api/student/{sid}`
```
Request: DELETE http://localhost:8080/api/student/1
Response: (No content)
```

## Data Flow

```
┌─────────────────────────┐
│    Frontend/Client      │
└────────────┬────────────┘
             │
             │ HTTP Request
             ▼
┌─────────────────────────┐
│  StudentController      │ (REST Endpoints)
│  @RestController        │
│  @RequestMapping        │
│  @CrossOrigin(*)        │
└────────────┬────────────┘
             │
             │ Calls Service Methods
             ▼
┌─────────────────────────┐
│  StudentService         │ (Business Logic)
│  (Interface)            │
└────────────┬────────────┘
             │
             │ Implements
             ▼
┌─────────────────────────┐
│  StudentServiceImpl      │ (Implementation)
│  @Service               │
└────────────┬────────────┘
             │
             │ Uses Repository
             ▼
┌─────────────────────────┐
│  StudentRepository      │ (Data Access)
│  extends CrudRepository │
└────────────┬────────────┘
             │
             │ Database Operations
             ▼
┌─────────────────────────┐
│  PostgreSQL Database    │
│  (student table)        │
└─────────────────────────┘
```

## Database Table

```sql
CREATE TABLE student (
    sid INTEGER PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
    name VARCHAR(255),
    college_name VARCHAR(255),
    phone_no BIGINT NOT NULL
)
```
