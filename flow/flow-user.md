# User Flow

## Overview
The User module manages users in the placement system including placement officers, HR managers, training coordinators, and other staff members who manage the placement process.

## Entity Model
```
Users
├── id (Long) - Primary Key, Auto-generated
├── name (String) - User's full name
├── email (String) - User's email address
└── designation (String) - User's job designation
```

## API Endpoints

### 1. Get All Users
**Endpoint:** `GET /api/users`
```
Request: GET http://localhost:8080/api/users
Response: 
[
  {
    "id": 1,
    "name": "Ananya Krishnan",
    "email": "ananya.k@gmail.com",
    "designation": "Placement Officer"
  },
  ...
]
```

### 2. Create User
**Endpoint:** `POST /api/users`
```
Request: POST http://localhost:8080/api/users
Body: {
  "name": "John Manager",
  "email": "john.m@gmail.com",
  "designation": "HR Manager"
}

Response:
{
  "id": 6,
  "name": "John Manager",
  "email": "john.m@gmail.com",
  "designation": "HR Manager"
}
```

### 3. Update User
**Endpoint:** `PUT /api/users`
```
Request: PUT http://localhost:8080/api/users
Body: {
  "id": 1,
  "name": "Ananya K",
  "email": "ananya.krishnan@gmail.com",
  "designation": "Senior Placement Officer"
}

Response:
{
  "id": 1,
  "name": "Ananya K",
  "email": "ananya.krishnan@gmail.com",
  "designation": "Senior Placement Officer"
}
```

### 4. Delete User
**Endpoint:** `DELETE /api/users/{id}`
```
Request: DELETE http://localhost:8080/api/users/1
Response: (No content)
```

## Data Flow

```
┌─────────────────────────┐
│    Frontend/Client      │
└────────────┬────────────┘
             │
             │ HTTP Request
             ▼
┌─────────────────────────┐
│  UsersController        │ (REST Endpoints)
│  @RestController        │
│  @RequestMapping        │
│  @CrossOrigin(*)        │
└────────────┬────────────┘
             │
             │ Calls Service Methods
             ▼
┌─────────────────────────┐
│  UsersService           │ (Business Logic)
│  (Interface)            │
└────────────┬────────────┘
             │
             │ Implements
             ▼
┌─────────────────────────┐
│  UsersServiceImpl        │ (Implementation)
│  @Service               │
└────────────┬────────────┘
             │
             │ Uses Repository
             ▼
┌─────────────────────────┐
│  UsersRepository        │ (Data Access)
│  extends JpaRepository  │
└────────────┬────────────┘
             │
             │ Database Operations
             ▼
┌─────────────────────────┐
│  PostgreSQL Database    │
│  (users table)          │
└─────────────────────────┘
```

## Database Table

```sql
CREATE TABLE users (
    id BIGINT PRIMARY KEY GENERATED BY DEFAULT AS IDENTITY,
    name VARCHAR(255),
    email VARCHAR(255),
    designation VARCHAR(255)
)
```
